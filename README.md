# Caregiver Support (Demo-only Streamlit App)

A simple caregiver-support demo app built with Streamlit.
- ✅ Works with **no OpenAI / API keys**
- ✅ Uses **rule-based logic** only (keywords + regex)
- ✅ **Non-diagnostic**: no medical diagnosis, no medication change advice
- ✅ Deployable on **Streamlit Community Cloud (free)**

## Run locally

1) Install Python 3.10+ (3.11 recommended)

2) Install dependencies:
```bash
pip install -r requirements.txt
